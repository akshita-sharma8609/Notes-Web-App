# Notes-Web-App

This is the notes web app using HTML, CSS, JAVASCRIPT AND BOOTSTRAP.

# Features:

- has a functionality to add notes with their title
- has a functionality to delete notes
- user can search for a specific note by typing either its title or any of the word that matches with the content of the note that is to be searched

# Website visiting link:

[Click here](https://akshita-sharma8609.github.io/Notes-Web-App/index.html) to visit the website

# How to run it on your local system?

1. Download the files from the repository
2. Run index.html on your local host
